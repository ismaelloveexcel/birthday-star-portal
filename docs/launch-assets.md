## 5 Objections And Suggested Replies

1. "Can I edit it later?"
   - Not yet. V1 creates one portal link from the details entered before checkout. If something important is wrong, contact support and I will help manually.

2. "Is this private?"
   - The portal link contains the invite details needed for guests to open it. Only include details you are comfortable sharing with invited guests.

3. "Do guests need to download anything?"
   - No. Guests open the link in their browser. It is designed for mobile sharing.

4. "What if checkout does not return me to the portal?"
   - Save the recovery code before payment. If the browser handoff fails, paste that code on the success page to rebuild the link.

5. "Can I track RSVPs?"
   - Not in v1. The portal includes RSVP actions that open WhatsApp or email, but there is no dashboard or guest tracking.
