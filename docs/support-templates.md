# Support Templates

Use these replies for the highest-probability paid-support cases.

## Lost portal link

Hi,

Thanks for your order. If you still have the recovery code shown before payment, open the success page and paste it there to rebuild your portal link instantly:

`https://yourdomain.com/success`

If you no longer have the recovery code, reply with:

- your payment receipt
- your child's name
- the party date
- the email address or phone number used during checkout

I will help you restore the link manually.

Best,
Wandering Dodo Support

## Browser storage / checkout handoff issue

Hi,

This usually happens when checkout opens in a different browser, an in-app browser, or a private session.

Please try this flow:

1. Open the site in Safari or Chrome.
2. Fill the form again.
3. Save the recovery code shown before payment.
4. Complete payment in that same browser.

If the portal link still does not appear after payment, paste the recovery code on:

`https://yourdomain.com/success`

If you still get stuck, reply with your receipt and I will sort it manually.

Best,
Wandering Dodo Support
